# 인터프리터 언어 : 순차적으로 실행되고 즉시 결과를 반환하는 프로그램 언어

# 호이스팅 : 변수와 함수의 메모리 공간을 선언 전에 *미리 할당하는 것*을 의미하여 선언 위치보다 먼저 호출 가능하다.

# Variables : const, let, var를 의미한다.

# const : 상수, 선언 후 값을 바꿀 수 없다.

# let : 변수, 생성 후에 값을 바꿀 수 있다.

# var : 변수, 선언 후 값을 변경할 수 있다는 것은 let과 동일하나, let과 달리 재선언할 수 있다는 차이가 있다.

# undefined : 변수는 선언했지만 값을 할당하지는 않음 , variable이 존재하지만 값이 주어지지 않은 상태

# null : 변수에 null(값이 없다)이 할당된다, null은 절대 자연적으로 발생하지 않음
